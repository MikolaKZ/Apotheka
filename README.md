### Дипломный проект курса LearnPython 11: Веб-приложение с делегированием части функционала в Telegram-бот

[подробное описание проекта](https://docs.google.com/document/d/1tNtsoLXqKW26wzFP6Q-3auhoIBqnZ6NUUd3ymyxyev8/edit)

### Краткое описание:

Apotheka - Приложение в котором пользователь может получить информацию по запрашиваемому лекарству, его стоимости, 
наличию в ближайшей аптеке, краткому описанию, а так же возможностью оповещения о приеме.
 
### Технологии

Apotheka использует следующие технологии:
* [FLASK](http://flask.pocoo.org/)
* [FLASK-SQLAlchemy](http://flask-sqlalchemy.pocoo.org/2.3/)


#### Запуск проекта на локальном сервере


1. ***Запустите виртуальное окружение***
for Windows/Linux
```sh
env\Scripts\Activate
```

```sh
source env/bin/activate
```
2. ***Установите все необходимые модули***
```sh
pip install -r requirements.txt
```

3. ***Запуск сервера Flask***
for Windows/Linux 
```sh
set or export FLASK_APP=webapp 
flask run
```
4. ***Введите адрес в браузере http://127.0.0.1:5000***
